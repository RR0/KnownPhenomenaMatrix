Web Graphics Library. : librairie graphique pour le web.

Besoin

Fournir une bibliothèque de fonctions graphiques 2D et 3D pour le Web déléguant des calculs à la carte graphique .

Analyse

Gestion de :

  • la caméra
  • la rotation 3D d'objets
  • le remplissage de facettes
  • les textures
  • la lumière

Conception

WebGL est basé sur OpenGL ES

Il permet de faire de rendus à l'aide de 2 types de shaders :

  • de vertex qui va fournir les coordonnées dans l'espace de projection (clipspace)
  • de pixels (de fragment) qui va fournir les couleurs.

pBuffers

pipeline model.
