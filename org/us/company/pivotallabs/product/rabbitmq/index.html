Rabbit Message Broker.

Analyse

RabbitMQ supporte divers protocoles de messagerie :

  • AMQP
    • 0.8
    • 0.9
    • 0.9.1
    • 1.0
  • STOMP
  • MQTT
  • HTTP via les plugins de :
    • management
    • Web-STOMP
    • JSON-RPC

Conception

RabbitMQ est composé de :

  • un serveur (standalone ou en cloud)
  • un client (Java/JMS, .Net, Erlang)

RabbitMQ est extensible via des plugins.
