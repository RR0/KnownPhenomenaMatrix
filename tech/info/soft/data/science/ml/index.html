Machine Learning : apprentissage automatique.

Motivation

Extraire un modèle (prédictif) d'un ensemble de données (model fitting).

Analyse

Un ensemble de données est utilisé pour "entraîner".

  • Apprentissage supervisé (supervised learning) : on indique à la machine ce qu'on attend d'elle, de manière à ce qu'elle produise une prédiction :
    • quantitative : fonction continue issue d'une régression linéaire
    • qualitative : classification via régression logistique
    • les deux : déterminant une probabilité d'appartenir à une classe
  • Apprentissage non supervisé (unsupervised learning) : on demande à la machine de trouver elle-même :
    • des groupes (clusters) de données

Conception

Il existe diverses techniques de ML mais on peut distinguer celles qui mobilisent une hiérarchie de "couches" d'apprentissage, qui constituent autant de niveau d'abstraction. On parlera alors d'apprentissage profond (deep learning).

  • Réseaux de neurones

Implémentation

Il peut bien sûr arriver qu'un algorithme d'apprentissage ne donne pas les résultats escomptés. On pourra alors opter pour différentes stratégies pour essayer de l'améliorer :

  • Entraîner l'algorithme sur plus de données, pour qu'il acquière une vision plus fine du problème ;
  • Ajouter ou enlever des caractéristiques du problème, qui pourraient manquer ou au contraire le complexifier inutilement ;
  • Ajouter des caractéristiques polynomiales (`x_1^2, x_2^2, x_1x_2`, etc.) si c'est adapté à votre algorithme, qui permettront de mieux cerner les hypothèses ;
  • Augmenter ou diminuer le facteur de régularisation, afin de plus ou moins prendre en compte des caractéristiques.

Cependant le travail nécessaire à ces améliorations peut se révéler long et il est recommandé de mener au préalable une analyse/un diagnostic aidant à sélectionner l'option la plus prometteuse :

  • Segmentation de vos données en 70% (aléatoire) pour l'ensemble d'entraînement (training set) et 30% pour l'ensemble de test (test set), et appliquer la fonction de calcul de coût du test sur l'ensemble d'entraînement.
