Analyse

Un ensemble de données est utilisé pour "entraîner".

  • Apprentissage supervisé (supervised learning) : on indique à la machine ce qu'on attend d'elle, de manière à ce qu'elle produise une prédiction :
    • quantitative : fonction continue issue d'une régression linéaire
    • qualitative : classification
    • les deux : régression logistique déterminant une probabilité d'appartenir à une classe
  • Apprentissage non supervisé (unsupervised learning) : on demande à la machine de trouver elle-même :
    • des groupes (clusters) de données
