Machine Learning : apprentissage automatique (ou apprentissage machine ou apprentissage statistique) .

Motivation

Extraire un modèle prédictif (model fitting) d'une analyse de données.

Analyse

Le ML est un domaine de l'IA où un ensemble de données est utilisé pour "entraîner". On distingue les apprentissages :

  • supervisé (supervised learning) : on indique à la machine ce qu'on attend d'elle (typiquement des résultats connus de données passées), de manière à ce qu'elle produise une prédiction :
    • quantitative : fonction continue issue d'une régression linéaire
    • qualitative : classification via régression logistique
    • les deux : déterminant une probabilité d'appartenir à une classe
    • SVM
    • Random Forest
  • non supervisé (unsupervised learning) : on demande à la machine de trouver elle-même des groupes (clusters) de données :
    • K-moyennes
  • par transfert (transfer learning) : réutiliser les connaissances acquises pour un problème pour les réutiliser sur un autre problème similaire :
    • Réseaux logiques de Markov
    • Réseaux bayésiens
  • par renforcement (Reinforcement Learning (RL)) : apprendre par tâtonnement (essais/erreurs) récompensés en cas de succès :
    • TD-learning

Conception

Il existe diverses techniques de ML mais on peut distinguer celles qui mobilisent une hiérarchie de "couches" d'apprentissage, qui constituent autant de niveau d'abstraction. On parlera alors d'apprentissage profond (deep learning).

L'efficacité du modèle choisi devra ensuite être évaluée.
