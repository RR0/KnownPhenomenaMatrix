Entrepôts Clés-Valeurs.

Besoin

Accéder rapidement à une données persistante.

Analyse

Un ECV permet de récupérer une donnée de structure quelconque (objet, grappe d'objet liés, fichier video ou audio, nombre, etc.). L'inconvénient de cela est que les risques de conflits entre clés augmentent.

Conception

Un ECV est similaire à une tâble de hâchage.

Pour pallier le problème de conflits entre clés, certains ECV proposent de partitionner leur espace de stockage selon le type des valeurs.

Exemples

Des exemples d'ECV sont :

  • Standalone
    • Redis
  • Distribués
    • Dynamo
    • Voldemort
    • Riak
