Shell : coque (d'un OS)

Besoin

IHM permettant d'accéder aux fonctions d'un OS.

Conception

Un invite de commande (prompt) de saisir diverses commandes avec des arguments variables.

Exemples

Des exemples de shells sont :

  • Unix
  • Windows
    • Command.com
    • cmd.exe
    • Powershell / Microsoft Command Shell (msh) / monad
  • Apple
    • Mac OS
    • iOS
