Platform.

Besoin

Constituer un environnement d'exécution.

Analyse

Une plate-forme peut exister à différents niveaux :

  • physique : une machine
  • logicielle
    • native : un OS ou une plate-forme tournant sur un OS
    • virtuelle : plate-forme logicielle s'exécutant sur plusieurs plates-formes sous-jacentes (natives typiquement). On parle alors de solution multi-plateformes (cross-platform)

Exemples

Des exemples de plates-formes sont :

  • natives
    • Solaris 8 sur Sun SPARC
    • Windows 2000 sur Intel Pentium 4
    • Mobiles
  • virtuelles
    • Java
    • Mobiles
      • Android
