Besoin

Unité d'éxécution d'une plate-forme.

Conception

Un processus peut être divisé en plusieurs sous-unités d'exécution parallèles : les threads.

Exemples

Des exemples de processus sont :

  • le processus d'un serveur Web
  • le processus d'une JVM exécutant une application Java

Notes

  • Le terme processus est parfois utilisé pour désigner une méthodologie de développement.
